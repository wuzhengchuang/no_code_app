## 集成测试场景

1. **离线使用流程**: 登录桌面应用 → 创建项目 → 编辑内容 → 断开网络 → 继续编辑 → 重新连接 → 自动同步变更

2. **本地代码生成**: 创建完整项目 → 本地生成代码 → 直接打开 VS Code → 验证代码可正常运行

3. **数据冲突解决**: 同时在 Web 端和桌面端编辑同一项目 → 桌面端离线修改 → Web 端修改 → 桌面端上线 → 同步时检测冲突 → 用户选择版本或合并
