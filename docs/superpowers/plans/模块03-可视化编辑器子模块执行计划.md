## 集成测试场景

1. **组件拖拽流程**: 从面板拖拽组件到画布 → 选中组件 → 修改属性 → 拖拽移动位置 → 嵌套到容器 → 撤销操作 → 重做

2. **页面管理流程**: 创建多个页面 → 设置主页 → 排序页面 → 编辑页面内容 → 查看组件树结构

3. **多组件协同**: 添加表单组件 → 添加输入框、按钮 → 配置按钮点击事件 → 验证组件树关系正确
