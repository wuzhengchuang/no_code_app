## 集成测试场景

1. **完整预览流程**: 编辑页面 → 打开预览 → 验证组件渲染 → 测试交互逻辑 → 切换设备尺寸 → 验证响应式布局

2. **API 调用预览**: 配置 API → 在预览中触发调用 → 查看网络请求面板 → 验证响应正确 → 切换 Mock 模式验证

3. **快照保存恢复**: 预览中设置特定状态 → 保存快照 → 继续修改 → 恢复快照 → 验证状态正确恢复
