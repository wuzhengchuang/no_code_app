## 集成测试场景

1. **完整 API 配置流程**: 创建数据模型 → 创建 API 配置 → 绑定响应 Schema → 生成 Mock 数据 → 测试 Mock API → 切换到真实 API 测试

2. **Mock 数据生成**: 定义复杂 JSON Schema → 自动生成 Mock 数据 → 验证数据符合 Schema → 自定义修改 Mock 数据

3. **多环境配置**: 配置开发/测试/生产环境变量 → 切换环境 → 验证 API 调用正确使用对应环境配置
