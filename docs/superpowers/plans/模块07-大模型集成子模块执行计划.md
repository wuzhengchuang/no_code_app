## 集成测试场景

1. **AI 生成组件**: 用户描述"一个带提交按钮的登录表单" → AI 生成组件配置 → 预览效果 → 用户确认应用

2. **多轮对话优化**: 生成页面 → 用户反馈"标题太小" → AI 修改配置 → 重新生成 → 用户满意

3. **API 智能生成**: 描述"用户列表接口,支持分页、搜索" → AI 生成 API 配置 + 响应 Schema + Mock 数据
