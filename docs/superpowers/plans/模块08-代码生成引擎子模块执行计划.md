## 集成测试场景

1. **完整代码生成流程**: 创建完整项目 (多个页面、API、逻辑) → 选择微信小程序 → 生成代码 → 下载 ZIP → 验证代码结构正确 → 可在开发者工具中运行

2. **多平台对比**: 同一项目 → 分别生成 H5 和微信小程序 → 验证两个平台功能一致 → 样式适配各自平台

3. **增量生成**: 修改某个页面 → 增量生成该页面代码 → 验证不影响其他页面
