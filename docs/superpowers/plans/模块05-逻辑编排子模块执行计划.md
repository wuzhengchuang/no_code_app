## 集成测试场景

1. **表单提交流程**: 表单提交按钮点击 → 验证表单数据 → 调用 API → 显示成功/失败提示 → 跳转页面

2. **条件分支流程**: 点击按钮 → 判断变量值 → 根据条件执行不同分支 → 更新对应组件

3. **循环处理流程**: 获取列表数据 → 循环遍历 → 逐条处理 → 更新列表组件
