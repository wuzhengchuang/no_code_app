## 集成测试场景

1. **团队协作流程**: 用户A创建团队 → 邀请用户B → 用户B接受 → 用户A创建团队项目 → 用户B编辑项目 → 验证权限正确 → 查看操作日志

2. **项目分享流程**: 用户A创建项目 → 生成分享链接 (密码保护, copy权限) → 用户B访问链接 → 输入密码 → 复制项目 → 验证新项目归属用户B

3. **权限级联测试**: 创建团队 → 添加成员 → 创建团队项目 → 修改成员团队角色 → 验证项目权限自动更新
