## 集成测试场景

1. **完整文档生成**: 创建包含多个页面、API、逻辑的项目 → 生成全套文档 → 验证 README 描述准确 → 验证 API 文档完整 → 验证 CLAUDE.md 规范正确

2. **AI 增强文档**: 选择一个复杂项目 → 使用 AI 生成功能描述 → 验证描述准确且易于理解 → 人工调整 → 重新生成

3. **文档版本管理**: 生成 v1.0 文档 → 修改项目 → 生成 v2.0 文档 → 对比差异 → 恢复 v1.0
