# 子模块 02: 项目管理子模块设计文档

## 概述

负责项目创建、编辑、删除、分享、导入导出、快照管理。提供完整的项目生命周期管理，支持个人项目和团队项目，提供版本控制、数据导入导出、协作分享等核心功能。

## 技术栈

- 后端: Python + FastAPI
- 数据库: MySQL 8.0+
- ORM: SQLAlchemy 2.0+
- 数据验证: Pydantic
- 文件存储: 本地文件系统 / OSS / S3
- 序列化: JSON
- 压缩: zipfile

## 数据模型

### projects 表

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BIGINT | PRIMARY KEY, AUTO_INCREMENT | 项目ID |
| name | VARCHAR(100) | NOT NULL | 项目名称 |
| description | TEXT | NULLABLE | 项目描述 |
| thumbnail_url | VARCHAR(500) | NULLABLE | 缩略图URL |
| owner_id | BIGINT | NOT NULL, FOREIGN KEY → users.id | 所有者ID |
| team_id | BIGINT | NULLABLE, FOREIGN KEY → teams.id | 团队ID |
| template_type | VARCHAR(20) | NOT NULL, DEFAULT 'blank' | 模板类型: blank/from_template |
| template_id | BIGINT | NULLABLE | 模板ID（如果从模板创建） |
| target_platforms | JSON | NOT NULL | 目标平台数组 |
| status | VARCHAR(20) | NOT NULL, DEFAULT 'draft' | 状态: draft/active/archived |
| config | JSON | NULLABLE | 项目全局配置 |
| project_data | JSON | NULLABLE | 完整项目数据（组件、页面等） |
| version | INT | NOT NULL, DEFAULT 1 | 版本号（乐观锁） |
| last_edited_by | BIGINT | NULLABLE, FOREIGN KEY → users.id | 最后编辑者ID |
| last_edited_at | DATETIME | NULLABLE | 最后编辑时间 |
| created_at | DATETIME | NOT NULL, DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| updated_at | DATETIME | NOT NULL, DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP | 更新时间 |

**索引**:
- `idx_projects_owner_id` (owner_id) - 普通索引，查询用户的项目
- `idx_projects_team_id` (team_id) - 普通索引，查询团队的项目
- `idx_projects_status` (status) - 普通索引，按状态筛选
- `idx_projects_created_at` (created_at) - 普通索引，按时间排序

**状态说明**:
- `draft`: 草稿，项目刚创建，未发布
- `active`: 活跃，项目正在编辑中
- `archived`: 已归档，项目不再编辑

**目标平台说明**:
```json
["wechat_miniapp", "alipay_miniapp", "h5", "react_native", "flutter", "uni_app"]
```

**索引**:
- `idx_snapshots_project_id` (project_id) - 普通索引，查询项目的快照
- `idx_snapshots_created_at` (created_at) - 普通索引，按时间排序

### project_shares 表

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BIGINT | PRIMARY KEY, AUTO_INCREMENT | 分享ID |
| project_id | BIGINT | NOT NULL, FOREIGN KEY → projects.id | 项目ID |
| share_token | VARCHAR(100) | UNIQUE, NOT NULL | 分享令牌 |
| permission | VARCHAR(20) | NOT NULL, DEFAULT 'view' | 权限: view/copy/fork |
| expires_at | DATETIME | NULLABLE | 过期时间 |
| is_password_protected | BOOLEAN | NOT NULL, DEFAULT FALSE | 是否密码保护 |
| password_hash | VARCHAR(255) | NULLABLE | 密码哈希 |
| view_count | INT | NOT NULL, DEFAULT 0 | 查看次数 |
| copy_count | INT | NOT NULL, DEFAULT 0 | 复制次数 |
| max_views | INT | NULLABLE | 最大查看次数限制 |
| max_copies | INT | NULLABLE | 最大复制次数限制 |
| created_by | BIGINT | NOT NULL, FOREIGN KEY → users.id | 创建者ID |
| is_active | BOOLEAN | NOT NULL, DEFAULT TRUE | 是否有效 |
| created_at | DATETIME | NOT NULL, DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| updated_at | DATETIME | NOT NULL, DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP | 更新时间 |

**索引**:
- `idx_shares_project_id` (project_id) - 普通索引，查询项目的分享
- `idx_shares_token` (share_token) - 唯一索引，通过令牌查找
- `idx_shares_expires_at` (expires_at) - 普通索引，清理过期分享

**权限说明**:
- `view`: 仅查看，不能复制或复刻
- `copy`: 可查看和复制项目
- `fork`: 可查看、复制和复刻项目

## API 接口

### 通用响应格式

**成功响应**:
```json
{
  "success": true,
  "data": {},
  "message": "操作成功"
}
```

**错误响应**:
```json
{
  "success": false,
  "error": {
    "code": "ERROR_CODE",
    "message": "错误描述",
    "details": {}
  }
}
```

**错误码说明**:
- `VALIDATION_ERROR`: 请求参数验证失败
- `AUTHENTICATION_FAILED`: 认证失败
- `AUTHORIZATION_FAILED`: 权限不足
- `PROJECT_NOT_FOUND`: 项目不存在
- `SNAPSHOT_NOT_FOUND`: 快照不存在
- `SHARE_NOT_FOUND`: 分享不存在
- `SHARE_EXPIRED`: 分享已过期
- `SHARE_PASSWORD_REQUIRED`: 需要分享密码
- `SHARE_PASSWORD_INVALID`: 分享密码错误
- `SHARE_LIMIT_EXCEEDED`: 分享次数超限
- `VERSION_CONFLICT`: 版本冲突
- `INVALID_TEMPLATE`: 无效的模板
- `IMPORT_FAILED`: 导入失败
- `EXPORT_FAILED`: 导出失败

### POST /api/projects

创建项目

**请求头**:
- `Authorization: Bearer {token}`

**请求体**:
```json
{
  "name": "新项目",
  "description": "项目描述",
  "templateType": "blank",
  "templateId": null,
  "targetPlatforms": ["wechat_miniapp", "h5"],
  "teamId": null
}
```

**请求验证**:
- `name`: 必填，1-100字符
- `description`: 可选，最大1000字符
- `templateType`: 可选，默认 blank，枚举值: blank/from_template
- `templateId`: template_type 为 from_template 时必填
- `targetPlatforms`: 必填，至少一个平台，最多支持 6 个平台
- `teamId`: 可选，团队项目时填写

**响应**:
```json
{
  "success": true,
  "data": {
    "id": 1,
    "name": "新项目",
    "description": "项目描述",
    "thumbnailUrl": null,
    "ownerId": 1,
    "teamId": null,
    "templateType": "blank",
    "targetPlatforms": ["wechat_miniapp", "h5"],
    "status": "draft",
    "version": 1,
    "createdAt": "2026-05-09T00:00:00Z",
    "updatedAt": "2026-05-09T00:00:00Z"
  },
  "message": "创建成功"
}
```

**可能的错误**:
- `VALIDATION_ERROR`: 参数验证失败
- `INVALID_TEMPLATE`: 模板不存在或无效

### GET /api/projects

获取项目列表

**请求头**:
- `Authorization: Bearer {token}`

**查询参数**:
- `status`: 可选，过滤状态: draft/active/archived
- `teamId`: 可选，团队ID筛选
- `page`: 可选，页码，默认 1
- `pageSize`: 可选，每页数量，默认 20，最大 100
- `sortBy`: 可选，排序字段: created_at/updated_at/name
- `sortOrder`: 可选，排序方向: asc/desc，默认 desc
- `keyword`: 可选，关键词搜索（项目名称或描述）

**响应**:
```json
{
  "success": true,
  "data": {
```

```python
"""项目相关 schema"""
from pydantic import BaseModel, Field, field_validator
from datetime import datetime
from typing import Optional, List, Any, Dict
from enum import Enum


class ProjectStatus(str, Enum):
    DRAFT = "draft"
    ACTIVE = "active"
    ARCHIVED = "archived"


class TemplateType(str, Enum):
    BLANK = "blank"
    FROM_TEMPLATE = "from_template"


class TargetPlatform(str, Enum):
    WECHAT_MINIA = "wechat_miniapp"
    ALIPAY_MINIA = "alipay_miniapp"
    H5 = "h5"
    REACT_NATIVE = "react_native"
    FLUTTER = "flutter"
    UNI_APP = "uni_app"


class ProjectBase(BaseModel):
    name: str = Field(..., min_length=1, max_length=100, description="项目名称")
    description: Optional[str] = Field(None, max_length=1000, description="项目描述")


class ProjectCreate(ProjectBase):
    templateType: TemplateType = Field(TemplateType.BLANK, description="模板类型")
    templateId: Optional[int] = Field(None, description="模板ID")
    targetPlatforms: List[TargetPlatform] = Field(..., description="目标平台")
    teamId: Optional[int] = Field(None, description="团队ID")

    @field_validator('targetPlatforms')
    @classmethod
    def validate_platforms(cls, v):
        if len(v) == 0:
            raise ValueError('至少需要选择一个平台')
        if len(v) > 6:
            raise ValueError('最多支持6个平台')
        return v


class ProjectUpdate(BaseModel):
    name: Optional[str] = Field(None, min_length=1, max_length=100)
    description: Optional[str] = Field(None, max_length=1000)
```

```python
"""项目服务"""
from sqlalchemy.orm import Session
from typing import List, Optional, Dict, Any
from fastapi import HTTPException, status
from datetime import datetime
import uuid

from backend.src.models.project import Project, Page, PageState
from backend.src.models.user import User
from backend.src.models.team import Team, TeamMember
from backend.src.schemas.project import (
    ProjectCreate, ProjectUpdate, ProjectDataUpdate,
    ProjectCopy, ProjectStatus, TargetPlatform
)
from backend.src.services.snapshot_service import SnapshotService


class ProjectService:
    def __init__(self, db: Session):
        self.db = db
        self.snapshot_service = SnapshotService(db)

    def create(self, data: ProjectCreate, user_id: int) -> Project:
        """创建项目"""
        # 如果从模板创建，验证模板
        template_data = None
        if data.templateType == "from_template" and data.templateId:
            from backend.src.services.template_service import TemplateService
            template_service = TemplateService(self.db)
            template = template_service.get_template(data.templateId)
            if template:
                template_data = template.template_data

        # 创建项目
        project = Project(
            name=data.name,
            description=data.description,
            owner_id=user_id,
            team_id=data.teamId,
            template_type=data.templateType.value,
            template_id=data.templateId,
            target_platforms=[p.value for p in data.targetPlatforms],
            status=ProjectStatus.DRAFT.value,
            project_data=template_data
        )
        self.db.add(project)
        self.db.flush()
        self.db.commit()
        self.db.refresh(project)
        return project

    def get_by_id(self, project_id: int) -> Optional[Project]:
        """获取项目"""
        return self.db.query(Project).filter(Project.id == project_id).first()

    def list_projects(
        self,
        user_id: int,
        status: Optional[str] = None,
        team_id: Optional[int] = None,
        keyword: Optional[str] = None,
        page: int = 1,
        page_size: int = 20,
        sort_by: str = "created_at",
        sort_order: str = "desc"
    ) -> tuple[List[Project], int]:
        """获取项目列表"""
        query = self.db.query(Project)

        # 基础权限过滤：个人项目或团队项目或有直接权限的项目
        from backend.src.models.collaborator import ProjectCollaborator

        user_team_ids = [
            tm.team_id for tm in
            self.db.query(TeamMember).filter(TeamMember.user_id == user_id).all()
        ]

        # 构建查询条件
        conditions = []
        # 个人项目
        conditions.append(Project.owner_id == user_id)
        # 团队项目
        if user_team_ids:
            conditions.append(Project.team_id.in_(user_team_ids))
        # 有直接权限的项目
        collaborator_project_ids = [
            cp.project_id for cp in
            self.db.query(ProjectCollaborator).filter(ProjectCollaborator.user_id == user_id).all()
        ]
        if collaborator_project_ids:
            conditions.append(Project.id.in_(collaborator_project_ids))

        # 组合条件
        from sqlalchemy import or_
        query = query.filter(or_(*conditions))

        # 状态过滤
        if status:
            query = query.filter(Project.status == status)

        # 团队过滤
        if team_id is not None:
            query = query.filter(Project.team_id == team_id)

        # 关键词搜索
        if keyword:
            from sqlalchemy import or_
            query = query.filter(
                or_(
                    Project.name.like(f"%{keyword}%"),
                    Project.description.like(f"%{keyword}%")
                )
            )

        # 排序
        sort_column = getattr(Project, sort_by, Project.created_at)
        if sort_order == "desc":
            sort_column = sort_column.desc()
        query = query.order_by(sort_column)

        # 总数
        total = query.count()

        # 分页
        offset = (page - 1) * page_size
        items = query.offset(offset).limit(page_size).all()

        return items, total

    def update_basic(
        self,
        project_id: int,
        data: ProjectUpdate,
        user_id: int
    ) -> Project:
        """更新项目基本信息"""
        project = self.get_by_id(project_id)
        if not project:
            raise HTTPException(
                status_code=status.HTTP_404_NOT_FOUND,
                detail="项目不存在"
            )

        # 更新字段
        update_data = data.model_dump(exclude_unset=True)
        if "targetPlatforms" in update_data:
            update_data["target_platforms"] = [
                p.value if isinstance(p, TargetPlatform) else p
                for p in update_data["targetPlatforms"]
            ]
            del update_data["targetPlatforms"]
        if "thumbnailUrl" in update_data:
            update_data["thumbnail_url"] = update_data["thumbnailUrl"]
            del update_data["thumbnailUrl"]

        for key, value in update_data.items():
            setattr(project, key, value)

        # 更新版本和编辑信息
        project.version += 1
        project.last_edited_by = user_id
        project.last_edited_at = datetime.utcnow()

        self.db.commit()
        self.db.refresh(project)
        return project

    def update_data(
        self,
        project_id: int,
        data: ProjectDataUpdate,
        user_id: int
    ) -> Project:
        """更新项目完整数据（使用乐观锁）"""
        project = self.get_by_id(project_id)
        if not project:
            raise HTTPException(
                status_code=status.HTTP_404_NOT_FOUND,
                detail="项目不存在"
            )

        # 检查版本
        if project.version != data.version:
            raise HTTPException(
                status_code=status.HTTP_409_CONFLICT,
                detail="版本冲突，数据已被其他用户修改"
            )

        project.project_data = data.projectData
        project.version += 1
        project.last_edited_by = user_id
        project.last_edited_at = datetime.utcnow()

        self.db.commit()
        self.db.refresh(project)
        return project

    def delete(self, project_id: int) -> None:
        """删除项目"""
        project = self.get_by_id(project_id)
        if not project:
            raise HTTPException(
                status_code=status.HTTP_404_NOT_FOUND,
                detail="项目不存在"
            )
        self.db.delete(project)
        self.db.commit()

    def copy(self, project_id: int, data: ProjectCopy, user_id: int) -> Project:
        """复制项目"""
        source = self.get_by_id(project_id)
        if not source:
            raise HTTPException(
                status_code=status.HTTP_404_NOT_FOUND,
                detail="项目不存在"
            )

        new_project = Project(
            name=data.name,
            description=source.description,
            thumbnail_url=source.thumbnail_url,
            owner_id=user_id,
            team_id=data.teamId,
            template_type="from_template",
            template_id=source.id,
            target_platforms=source.target_platforms,
            status=ProjectStatus.DRAFT.value,
            config=source.config,
            project_data=source.project_data
        )
        self.db.add(new_project)
        self.db.commit()
        self.db.refresh(new_project)
        return new_project

    def check_permission(
        self,
        project_id: int,
        user_id: int,
        required_permission: str = "read"
    ) -> bool:
        """检查用户对项目的权限"""
        project = self.get_by_id(project_id)
        if not project:
            return False

        # 所有者拥有所有权限
        if project.owner_id == user_id:
            return True

        # 检查团队权限
        if project.team_id:
            from backend.src.models.team import TeamMember

            team_member = self.db.query(TeamMember).filter(
                TeamMember.team_id == project.team_id,
                TeamMember.user_id == user_id
            ).first()

            if team_member:
                team_permissions = {
                    "owner": "admin",
                    "admin": "admin",
                    "member": "write",
                    "viewer": "read"
                }
                user_perm = team_permissions[team_member.role]
                if self._has_permission(user_perm, required_permission):
                    return True

        # 检查协作者权限
        from backend.src.models.collaborator import ProjectCollaborator

        collaborator = self.db.query(ProjectCollaborator).filter(
            ProjectCollaborator.project_id == project_id,
            ProjectCollaborator.user_id == user_id
        ).first()

        if collaborator:
            if self._has_permission(collaborator.permission, required_permission):
                return True

        return False

    def _has_permission(self, user_permission: str, required_permission: str) -> bool:
        """检查权限等级"""
        permission_levels = {
            "read": 1,
            "write": 2,
            "admin": 3
        }
        return permission_levels.get(user_permission, 0) >= permission_levels.get(required_permission, 0)
```

## 安全考虑

### 项目数据安全
- 严格的权限检查，每个接口都验证用户权限
- 分享链接使用随机生成的令牌，避免可猜测
- 可选的密码保护，增加安全性
- 支持过期时间和访问次数限制
- 分享内容只包含预览信息，不直接暴露完整数据

### 并发编辑安全
- 使用乐观锁（版本号）处理并发编辑冲突
- 每次更新递增版本号
- 版本不匹配时返回明确的错误提示

### 导入导出安全
- 导入文件验证格式和内容完整性
- 防止恶意文件导入
- 资源文件安全处理

### 数据保护
- 所有者拥有项目数据的完全控制权
- 删除项目时级联删除所有相关数据
- 快照包含完整数据，便于恢复
- 支持归档状态，保护历史项目不被误修改
