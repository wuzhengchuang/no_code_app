# 子模块 01: 用户与权限子模块设计文档

## 概述

负责用户注册登录、权限管理、团队协作功能。提供完整的认证授权体系，支持JWT令牌认证、多级权限控制、团队协作管理。

## 技术栈

- 后端: Python + FastAPI
- 数据库: MySQL 8.0+
- 认证: JWT (RFC 7519)
- 密码加密: bcrypt
- ORM: SQLAlchemy 2.0+
- 数据验证: Pydantic

## 数据模型

### users 表

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BIGINT | PRIMARY KEY, AUTO_INCREMENT | 用户ID |
| email | VARCHAR(255) | UNIQUE, NOT NULL | 邮箱 |
| password_hash | VARCHAR(255) | NOT NULL | 密码哈希(使用bcrypt) |
| nickname | VARCHAR(100) | NOT NULL | 昵称 |
| avatar_url | VARCHAR(500) | NULLABLE | 头像URL |
| status | TINYINT | NOT NULL, DEFAULT 1 | 状态: 0-禁用, 1-正常 |

**索引:**
- `idx_users_email` (email) - 唯一索引，用于登录查询
- `idx_users_status` (status) - 普通索引，用于状态筛选

### team_members 表

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BIGINT | PRIMARY KEY, AUTO_INCREMENT | 记录ID |
| team_id | BIGINT | NOT NULL, FOREIGN KEY → teams.id | 团队ID |
| user_id | BIGINT | NOT NULL, FOREIGN KEY → users.id | 用户ID |
| role | VARCHAR(20) | NOT NULL, DEFAULT 'member' | 角色: owner/admin/member/viewer |
| invited_by | BIGINT | NULLABLE, FOREIGN KEY → users.id | 邀请人ID |
| joined_at | DATETIME | NULLABLE | 加入时间 |

**索引:**
- `idx_team_members_team_id` (team_id) - 普通索引

**角色说明:**
- `owner`: 团队所有者，拥有所有权限，不可移除，只能转让

- `member`: 团队成员，可查看和编辑团队项目
- `viewer`: 只读成员，仅可查看团队项目

### 通用响应格式

**成功响应:**
```json
{
  "success": true,
  "data": {},
  "message": "操作成功"
}
```

**错误响应:**
```json
{
  "success": false,

    "code": "ERROR_CODE",

  }

```

**错误码说明:**
- `VALIDATION_ERROR`: 请求参数验证失败
- `AUTHENTICATION_FAILED`: 认证失败
- `AUTHORIZATION_FAILED`: 权限不足
- `USER_NOT_FOUND`: 用户不存在
- `USER_ALREADY_EXISTS`: 用户已存在
- `TEAM_NOT_FOUND`: 团队不存在
- `INVALID_CREDENTIALS`: 凭据无效
- `TOKEN_EXPIRED`: 令牌已过期
- `TOKEN_INVALID`: 令牌无效
- `RATE_LIMIT_EXCEEDED`: 请求频率超限

### POST /api/auth/register

用户注册

**请求体:**
```json
{
  "email": "user@example.com",
  "password": "password123",
  "nickname": "用户昵称"
}
```

**请求验证:**
- `email`: 必须是有效邮箱格式，最大255字符
- `password`: 最小8字符，必须包含字母和数字
- `nickname`: 最大100字符，不能为空

**响应:**
```json
{
  "success": true,
  "data": {
    "userId": 1,
    "email": "user@example.com",
    "nickname": "用户昵称",
    "createdAt": "2026-05-09T00:00:00Z"
  },
  "message": "注册成功"
}
```

**可能的错误:**
- `USER_ALREADY_EXISTS`: 邮箱已被注册
- `VALIDATION_ERROR`: 参数验证失败

### POST /api/auth/login

用户登录

**请求体:**
```json
{
  "email": "user@example.com",
  "password": "password123"
}
```

**响应:**
```json
{
  "success": true,
  "data": {
    "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    "refreshToken": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    "expiresAt": "2026-05-09T02:00:00Z",
    "refreshExpiresAt": "2026-05-16T00:00:00Z",
    "tokenType": "Bearer",
    "user": {
      "id": 1,
      "email": "user@example.com",
      "nickname": "用户昵称",
      "avatarUrl": null
    }
  },
  "message": "登录成功"
}
```

**JWT Payload 结构:**
```json
{
  "sub": "1",
  "email": "user@example.com",
  "type": "access",
  "iat": 1715203200,
  "exp": 1715210400,
  "jti": "unique-session-id"
}
```

**令牌有效期:**
- Access Token: 2小时
- Refresh Token: 7天

**可能的错误:**
- `INVALID_CREDENTIALS`: 邮箱或密码错误
- `USER_NOT_FOUND`: 用户不存在
- `VALIDATION_ERROR`: 参数验证失败

## 权限校验中间件

### FastAPI 依赖注入实现

```python
from fastapi import Depends, HTTPException, status, Request
from fastapi.security import HTTPBearer, HTTPAuthorizationCredentials
from sqlalchemy.orm import Session
from jose import JWTError, jwt
from datetime import datetime

from backend.src.db.session import get_db
from backend.src.models.user import User, UserSession
from backend.src.models.team import TeamMember
from backend.src.core.config import get_settings

settings = get_settings()
security = HTTPBearer()

async def get_current_user(
    credentials: HTTPAuthorizationCredentials = Depends(security),
    db: Session = Depends(get_db)
) -> User:
    """获取当前登录用户"""
    credentials_exception = HTTPException(
        status_code=status.HTTP_401_UNAUTHORIZED,
        detail="无法验证凭据",
        headers={"WWW-Authenticate": "Bearer"},
    )

    try:
        token = credentials.credentials
        payload = jwt.decode(token, settings.SECRET_KEY, algorithms=[settings.ALGORITHM])
        user_id: str = payload.get("sub")
        token_type: str = payload.get("type")
        jti: str = payload.get("jti")

        if user_id is None or token_type != "access":
            raise credentials_exception

    except JWTError:
        raise credentials_exception

    # 检查令牌是否已撤销
    session = db.query(UserSession).filter(
        UserSession.token == token,
        UserSession.is_revoked == False
    ).first()

    if not session or session.expires_at < datetime.utcnow():
        raise credentials_exception

    # 获取用户
    user = db.query(User).filter(User.id == int(user_id)).first()
    if user is None or user.status != 1:
        raise credentials_exception

    return user

def require_team_role(min_role: str):
    """要求团队角色权限"""
    def dependency(
        team_id: int,
        current_user: User = Depends(get_current_user),
        db: Session = Depends(get_db)
    ):
        # 角色优先级（从高到低）
        role_hierarchy = {
            'owner': 4,
            'admin': 3,
            'member': 2,
            'viewer': 1
        }

        # 查询用户在团队中的角色
        team_member = db.query(TeamMember).filter(
            TeamMember.team_id == team_id,
            TeamMember.user_id == current_user.id
        ).first()

        if not team_member:
            raise HTTPException(
                status_code=status.HTTP_403_FORBIDDEN,
                detail="不是团队成员"
            )

        # 检查角色权限是否足够
        if role_hierarchy[team_member.role] < role_hierarchy[min_role]:
            raise HTTPException(
                status_code=status.HTTP_403_FORBIDDEN,
                detail="权限不足"
            )

        return team_member

    return Depends(dependency)

def require_project_permission(required_permission: str):
    """要求项目权限"""
    def dependency(
        project_id: int,
        current_user: User = Depends(get_current_user),
        db: Session = Depends(get_db)
    ):
        # 权限优先级
        permission_hierarchy = {
            'admin': 3,
            'write': 2,
            'read': 1
        }

        # 首先检查是否有直接的项目权限
        from backend.src.models.user import UserProjectPermission

        project_permission = db.query(UserProjectPermission).filter(
            UserProjectPermission.project_id == project_id,
            UserProjectPermission.user_id == current_user.id
        ).first()

        # 如果有直接权限，检查是否足够
        if project_permission:
            if permission_hierarchy[project_permission.permission] >= permission_hierarchy[required_permission]:
                return project_permission

        # 检查项目是否属于某个团队，用户在团队中的权限
        from backend.src.models.project import Project

        project = db.query(Project).filter(Project.id == project_id).first()
        if not project:
            raise HTTPException(
                status_code=status.HTTP_404_NOT_FOUND,
                detail="项目不存在"
            )

        # 如果项目属于团队，检查团队成员角色
        if project.team_id:
            team_member = db.query(TeamMember).filter(
                TeamMember.team_id == project.team_id,
                TeamMember.user_id == current_user.id
            ).first()

            if team_member:
                # 团队 owner/admin/member 默认有 write 权限，viewer 有 read 权限
                team_role_permissions = {
                    'owner': 'admin',
                    'admin': 'admin',
                    'member': 'write',
                    'viewer': 'read'
                }
                user_permission = team_role_permissions[team_member.role]
                if permission_hierarchy[user_permission] >= permission_hierarchy[required_permission]:
                    return True

        # 如果项目属于当前用户，有所有权限
        if project.owner_id == current_user.id:
            return True

        # 没有权限
        raise HTTPException(
            status_code=status.HTTP_403_FORBIDDEN,
            detail="项目权限不足"
        )

    return Depends(dependency)

# 便捷依赖
async def get_current_active_user(
    current_user: User = Depends(get_current_user)
) -> User:
    """获取当前活跃用户"""
    if current_user.status != 1:
        raise HTTPException(
            status_code=status.HTTP_403_FORBIDDEN,
            detail="用户已被禁用"
        )
    return current_user
```

### 权限决策矩阵

#### 团队操作权限

| 操作 | owner | admin | member | viewer |
|------|-------|-------|--------|--------|

```python
"""安全相关工具"""
from passlib.context import CryptContext
from jose import jwt
from datetime import datetime, timedelta
import uuid
from typing import Optional, Tuple
from backend.src.core.config import get_settings

settings = get_settings()
pwd_context = CryptContext(schemes=["bcrypt"], deprecated="auto")

def verify_password(plain_password: str, hashed_password: str) -> bool:
    """验证密码"""
    return pwd_context.verify(plain_password, hashed_password)

def get_password_hash(password: str) -> str:
    """获取密码哈希"""
    return pwd_context.hash(password)

def create_access_token(data: dict, expires_delta: Optional[timedelta] = None) -> Tuple[str, datetime]:
    """创建访问令牌"""
    to_encode = data.copy()
    if expires_delta:
        expire = datetime.utcnow() + expires_delta
    else:
        expire = datetime.utcnow() + timedelta(hours=2)

    to_encode.update({
        "exp": expire,
        "iat": datetime.utcnow(),
        "type": "access",
        "jti": str(uuid.uuid4())
    })
    encoded_jwt = jwt.encode(to_encode, settings.SECRET_KEY, algorithm=settings.ALGORITHM)
    return encoded_jwt, expire

def create_refresh_token(data: dict, expires_delta: Optional[timedelta] = None) -> Tuple[str, datetime]:
    """创建刷新令牌"""
    to_encode = data.copy()
    if expires_delta:
        expire = datetime.utcnow() + expires_delta
    else:
        expire = datetime.utcnow() + timedelta(days=7)

    to_encode.update({
        "exp": expire,
        "iat": datetime.utcnow(),
        "type": "refresh",
        "jti": str(uuid.uuid4())
    })
    encoded_jwt = jwt.encode(to_encode, settings.SECRET_KEY, algorithm=settings.ALGORITHM)
    return encoded_jwt, expire

def decode_token(token: str) -> Optional[dict]:
    """解码令牌"""
    try:
        payload = jwt.decode(token, settings.SECRET_KEY, algorithms=[settings.ALGORITHM])
        return payload
    except JWTError:
        return None
```

#### services/auth_service.py
```python
"""认证服务"""
from sqlalchemy.orm import Session
from datetime import datetime, timedelta
from typing import Optional, Tuple
from fastapi import HTTPException, status
import re

from backend.src.models.user import User, UserSession
from backend.src.schemas.auth import UserRegister, UserLogin, TokenResponse, AuthResponse
from backend.src.schemas.user import UserResponse
from backend.src.core.security import (
    verify_password, get_password_hash,
    create_access_token, create_refresh_token
)
from backend.src.utils.jwt import hash_token

class AuthService:
    def __init__(self, db: Session):
        self.db = db

    def _validate_password(self, password: str) -> bool:
        """验证密码强度"""
        if len(password) < 8:
            return False
        if not re.search(r'[A-Za-z]', password):
            return False
        if not re.search(r'[0-9]', password):
            return False
        return True

    def register(self, data: UserRegister) -> AuthResponse:
        """用户注册"""
        # 验证密码强度
        if not self._validate_password(data.password):
            raise HTTPException(
                status_code=status.HTTP_400_BAD_REQUEST,
                detail="密码强度不足，至少8位，包含字母和数字"
            )

        # 检查用户是否已存在
        existing_user = self.db.query(User).filter(User.email == data.email).first()
        if existing_user:
            raise HTTPException(
                status_code=status.HTTP_409_CONFLICT,
                detail="用户已存在"
            )

        # 创建用户
        user = User(
            email=data.email,
            password_hash=get_password_hash(data.password),
            nickname=data.nickname,
            status=1
        )
        self.db.add(user)
        self.db.flush()  # 获取用户ID

        # 创建令牌
        token_data = {"sub": str(user.id), "email": user.email}
        access_token, access_expire = create_access_token(token_data)
        refresh_token, refresh_expire = create_refresh_token(token_data)

        # 创建会话记录
        session = UserSession(
            user_id=user.id,
            token=hash_token(access_token),
            refresh_token=hash_token(refresh_token),
            expires_at=access_expire,
            refresh_expires_at=refresh_expire
        )
        self.db.add(session)

        self.db.commit()
        self.db.refresh(user)

        # 构建响应
        user_response = UserResponse(
            id=user.id,
            email=user.email,
            nickname=user.nickname,
            avatarUrl=user.avatar_url,
            status=user.status,
            lastLoginAt=user.last_login_at,
            createdAt=user.created_at,
            updatedAt=user.updated_at
        )

        return AuthResponse(
            token=access_token,
            refreshToken=refresh_token,
            expiresAt=access_expire,
            refreshExpiresAt=refresh_expire,
            user=user_response
        )

    def login(self, data: UserLogin, ip_address: Optional[str] = None, user_agent: Optional[str] = None) -> AuthResponse:
        """用户登录"""
        # 查找用户
        user = self.db.query(User).filter(User.email == data.email).first()
        if not user:
            raise HTTPException(
                status_code=status.HTTP_401_UNAUTHORIZED,
                detail="邮箱或密码错误"
            )

        # 验证密码
        if not verify_password(data.password, user.password_hash):
            raise HTTPException(
                status_code=status.HTTP_401_UNAUTHORIZED,
                detail="邮箱或密码错误"
            )

        # 检查用户状态
        if user.status != 1:
            raise HTTPException(
                status_code=status.HTTP_403_FORBIDDEN,
                detail="用户已被禁用"
            )

        # 更新最后登录时间
        user.last_login_at = datetime.utcnow()

        # 识别设备类型
        device_type = self._detect_device_type(user_agent)

        # 创建令牌
        token_data = {"sub": str(user.id), "email": user.email}
        access_token, access_expire = create_access_token(token_data)
        refresh_token, refresh_expire = create_refresh_token(token_data)

        # 创建会话记录
        session = UserSession(
            user_id=user.id,
            token=hash_token(access_token),
            refresh_token=hash_token(refresh_token),
            expires_at=access_expire,
            refresh_expires_at=refresh_expire,
            ip_address=ip_address,
            user_agent=user_agent,
            device_type=device_type
        )
        self.db.add(session)

        self.db.commit()
        self.db.refresh(user)

        # 构建响应
        user_response = UserResponse(
            id=user.id,
            email=user.email,
            nickname=user.nickname,
            avatarUrl=user.avatar_url,
            status=user.status,
            lastLoginAt=user.last_login_at,
            createdAt=user.created_at,
            updatedAt=user.updated_at
        )

        return AuthResponse(
            token=access_token,
            refreshToken=refresh_token,
            expiresAt=access_expire,
            refreshExpiresAt=refresh_expire,
            user=user_response
        )

    def refresh_token(self, refresh_token: str) -> TokenResponse:
        """刷新令牌"""
        from backend.src.utils.jwt import decode_token

        # 解码刷新令牌
        payload = decode_token(refresh_token)
        if not payload or payload.get("type") != "refresh":
            raise HTTPException(
                status_code=status.HTTP_401_UNAUTHORIZED,
                detail="无效的刷新令牌"
            )

        # 查找会话记录
        hashed_refresh_token = hash_token(refresh_token)
        session = self.db.query(UserSession).filter(
            UserSession.refresh_token == hashed_refresh_token,
            UserSession.is_revoked == 0
        ).first()

        if not session:
            raise HTTPException(
                status_code=status.HTTP_401_UNAUTHORIZED,
                detail="刷新令牌不存在或已撤销"
            )

        if session.refresh_expires_at < datetime.utcnow():
            raise HTTPException(
                status_code=status.HTTP_401_UNAUTHORIZED,
                detail="刷新令牌已过期"
            )

        # 获取用户
        user = self.db.query(User).filter(User.id == session.user_id).first()
        if not user or user.status != 1:
            raise HTTPException(
                status_code=status.HTTP_401_UNAUTHORIZED,
                detail="用户不存在或已被禁用"
            )

        # 撤销旧会话
        session.is_revoked = 1

        # 创建新令牌
        token_data = {"sub": str(user.id), "email": user.email}
        new_access_token, new_access_expire = create_access_token(token_data)
        new_refresh_token, new_refresh_expire = create_refresh_token(token_data)

        # 创建新会话记录
        new_session = UserSession(
            user_id=user.id,
            token=hash_token(new_access_token),
            refresh_token=hash_token(new_refresh_token),
            expires_at=new_access_expire,
            refresh_expires_at=new_refresh_expire,
            ip_address=session.ip_address,
            user_agent=session.user_agent,
            device_type=session.device_type
        )
        self.db.add(new_session)
        self.db.commit()

        return TokenResponse(
            token=new_access_token,
            refreshToken=new_refresh_token,
            expiresAt=new_access_expire,
            refreshExpiresAt=new_refresh_expire
        )

    def logout(self, user_id: int, refresh_token: str) -> None:
        """用户登出"""
        hashed_refresh_token = hash_token(refresh_token)

        # 撤销所有该用户的活跃会话（或只撤销指定的会话）
        sessions = self.db.query(UserSession).filter(
            UserSession.user_id == user_id,
            UserSession.refresh_token == hashed_refresh_token,
            UserSession.is_revoked == 0
        ).all()

        for session in sessions:
            session.is_revoked = 1

        self.db.commit()

    def _detect_device_type(self, user_agent: Optional[str]) -> Optional[str]:
        """识别设备类型"""
        if not user_agent:
            return None

        ua_lower = user_agent.lower()
        if 'mobile' in ua_lower or 'android' in ua_lower or 'iphone' in ua_lower:
            return 'mobile'
        elif 'electron' in ua_lower:
            return 'desktop'
        else:
            return 'web'
```

## 安全考虑

### 密码安全
- 使用 bcrypt 算法哈希密码（工作因子 12）
- 不存储明文密码
- 密码强度要求：至少8位，包含字母和数字
- 修改密码时验证旧密码

### 令牌安全
- Access Token 有效期 2 小时
- Refresh Token 有效期 7 天
- 使用 HS256 算法签名
- 令牌在数据库中哈希存储（防止数据库泄露导致令牌滥用）
- Refresh Token 只能使用一次（使用后立即撤销旧的，生成新的）
- 支持主动撤销令牌

### 请求安全
- 所有认证接口使用 HTTPS
- 实现速率限制（防止暴力破解）
- 请求参数验证（Pydantic）
- SQL 注入防护（使用 SQLAlchemy ORM）

### 会话安全
- 记录设备信息（IP、User-Agent）
- 支持查看和撤销活跃会话
- 修改密码后撤销所有其他会话
- 检测异常登录（地理位置、设备变化）


## 部署考虑

### 环境变量
```env
# 应用配置
APP_NAME="NoCode Platform"
DEBUG=true

# 数据库配置
DATABASE_URL="mysql+pymysql://user:pass@host:3306/nocode"

# JWT 配置
SECRET_KEY="your-secret-key-here-change-in-production"
ALGORITHM="HS256"
ACCESS_TOKEN_EXPIRE_MINUTES=120
REFRESH_TOKEN_EXPIRE_DAYS=7

# CORS 配置
CORS_ORIGINS="http://localhost:3000,http://localhost:5173"
```
